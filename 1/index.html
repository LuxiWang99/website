My projects

SeGuard

SeGuard is a project that classifies Android applications. My role in this project is to take the semantic graphs from my supervisor, and parse(featurize) it into vectors. These vectors contains information (topological features) of the graph that we think is crucial for the classicication. Then, we feed it to the downstream random forest classifier. Up to now, we have about 70% of accuracy for binary classification (Answer the question of: is it a benign or malicious application).
